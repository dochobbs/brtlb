What brtlb is

brtlb is a pediatric AI scribe that runs entirely in your browser. Press one button, talk to the patient and family, get a finished note. Multi-patient sibling visits, well-child checks, ADHD follow-ups, behavioral health, sick visits — all handled with peds-tuned prompts.

It's yours. Your audio, your transcript, your note, your API keys — all of it stays on your device unless you choose to share. brtlb has no server, no account, no telemetry tied to a patient. Why we built it this way →

What's in the box

One-tap recording

Ambient mode (whole room, with speaker labels) or dictation mode (you, narrating). No new-visit forms, no patient picker.

Multi-patient visits

Sibling sick visits work out of the box. brtlb auto-detects each child and produces a separate note per kid.

Visit-type-aware templates

Well-child, sick visit, ADHD med check, behavioral health, autism eval. Auto-selected from the transcript, override any time.

Custom templates

Write your own template in plain English. Polish-with-AI gives you a clean version. How →

Speaker roles + diarization safety

If the transcription merges voices, brtlb flags it and proposes a one-click split. No silent misattribution.

Tweak with AI

Plain-English steering for any section of the note. "Shorten the HPI," "redo Plan in bullets," etc.

Patterns

Narrative, focused, telegraphic — adjusts style without changing facts. Mix with any template.

QA review + clinical pearls

Optional second pass that checks the note against the transcript and surfaces nuanced observations worth keeping.

Getting started in 5 minutes

  1. Get a Google Cloud API key for Gemini

    If your practice has Google Workspace, you almost certainly have a Google HIPAA BAA already. Create a Gemini API key in your Cloud project. BAA setup walkthrough →

  2. Sign the AssemblyAI BAA

    Five-minute DocuSign. Generate an API key from your AAI account.

  3. Paste both keys into Settings

    Open brtlb → Settings. Paste, save. brtlb tests them locally — no key ever leaves your browser to anyone but the provider you chose.

  4. Hit Record

    Mic permission once, then you're recording. Stop when you're done. Note appears on Review in seconds for a typical visit.

  5. Make it yours

    Custom templates, style preferences, save your favorite patterns. Customization guide →
