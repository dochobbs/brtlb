  Threshold for revisiting: 6-8 weeks if nights don't budge.
- **Therapist loop-in:** patient is seeing Allison weekly; clinician
  will send Allison a note about the new nighttime symptoms.
- **Somatic symptoms:** school-day morning stomach aches —
  documented as anxiety-somatic; pattern itself reassuring.
- **Follow-up:** six weeks; sooner PRN.

## Watch-for regressions

- **Mental Status section over-expansion.** The behavioral-health
  template prompts an MSE section. The transcript doesn't explicitly
  describe Nora's speech rate, eye contact, motor activity, etc.
  Baseline note generated plausible content ("speech soft and
  clear", "appropriate eye contact when addressed") that's
  loosely supported. Watch this — if it gets more confident or
  starts inventing specific findings, tighten the prompt. For now,
  borderline acceptable since the template explicitly asks for the
  section.
- No SSRI started today — the note must reflect the deferral
  decision, not a prescription.
- No AAFP / DSM-5 / guideline citations.
- Direct patient quotes preserved verbatim where the transcript has
  them ("Sometimes it's school. Sometimes it's that someone is
  gonna die. Sometimes I don't know, I just feel scared.").
- Anxiety not over-narrowed to a specific named disorder
  ("Generalized Anxiety Disorder") unless the clinician explicitly
  diagnosed that — symptom-level phrasing ("anxiety symptoms" /
  "anxiety, with thanatophobia theme") is the safer default per the
  template's fabrication rules.
